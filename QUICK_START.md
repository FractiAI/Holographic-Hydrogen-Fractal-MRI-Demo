## ğŸ› Troubleshooting

### Demo Won't Load
```bash
# Clear cache and reinstall
rm -rf node_modules package-lock.json
npm install
npm run dev
```

### Tesla AI Not Working
- Check `.env` file exists and has valid `VITE_GROQ_API_KEY`
- Verify API key at [console.groq.com](https://console.groq.com/keys)
- Demo works without Tesla AI (limited functionality)

### Slow Performance
- Close other browser tabs
- Update graphics drivers
- Use Chrome or Firefox (best performance)
- Reduce fractal depth in Stage 7 (depth 2 instead of 4)

### Blank Screen
- Check browser console (F12) for errors
- Ensure JavaScript is enabled
- Try different browser
- Verify WebGL support at [webglreport.com](https://webglreport.com)
