# Protocol IX: Reality Catalog System

**Streaming Catalog for Awareness Experiences**

**Classification:** EXPERIENCE_MARKETPLACE  
**Status:** OPERATIONAL  
**Integration:** Blockchain-Based Rental & Ownership  
**Content Type:** Pre-Packaged, Custom, AI-Guided Realities

---

## Abstract

The Reality Catalog is a browsable marketplace of awareness experiences where users select characters, edges (awareness contexts), missions, stories, and genres to create or consume immersive realities. All experiences are blockchain-tokenized with flexible access models: hourly rental, daily rental, monthly lease, or permanent deed ownership.

**Key Innovation:** Text-to-Reality engine converts natural language descriptions into fully navigable awareness experiences.

### 4.2 Smart Contract Details

```solidity
// Simplified example (actual contracts more complex)

contract RealityToken {
    enum AccessType { HOURLY, DAILY, MONTHLY, DEED }
    
    struct Reality {
        uint256 id;
        string metadataURI;
        address creator;
        uint256 hourlyPrice;
        uint256 dailyPrice;
        uint256 monthlyPrice;
        uint256 deedPrice;
        bool allowSubLease;
        bool allowModification;
    }
    
    struct Rental {
        uint256 realityId;
        address renter;
        AccessType accessType;
        uint256 expiryTimestamp;
        bool active;
    }
    
    mapping(uint256 => Reality) public realities;
    mapping(uint256 => mapping(address => Rental)) public rentals;
    mapping(uint256 => address) public deedOwners;
    
    function rentHourly(uint256 realityId, uint256 hours) external payable {
        require(msg.value >= realities[realityId].hourlyPrice * hours);
        rentals[realityId][msg.sender] = Rental({
            realityId: realityId,
            renter: msg.sender,
            accessType: AccessType.HOURLY,
            expiryTimestamp: block.timestamp + (hours * 1 hours),
            active: true
        });
        emit RentalCreated(realityId, msg.sender, hours);
    }
    
    function purchaseDeed(uint256 realityId) external payable {
        require(msg.value >= realities[realityId].deedPrice);
        require(deedOwners[realityId] == address(0)); // Not already owned
        deedOwners[realityId] = msg.sender;
        emit DeedPurchased(realityId, msg.sender);
    }
    
    function checkAccess(uint256 realityId, address user) public view returns (bool) {
        // Check deed ownership
        if (deedOwners[realityId] == user) return true;
        
        // Check active rental
        Rental memory rental = rentals[realityId][user];
        if (rental.active && rental.expiryTimestamp > block.timestamp) {
            return true;
        }
        
        return false;
    }
}
```
